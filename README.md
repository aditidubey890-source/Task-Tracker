# Docket — Task Tracker

A clean, elegant task tracking web app with a warm editorial aesthetic.

## Features
- ✅ Add, edit, complete & delete tasks
- 🗂 Categories: Work, Personal, Health, Finance
- 🔴 Priority levels: High, Medium, Low
- 📅 Due dates with overdue detection
- 🔍 Filter by priority, sort by date/name/priority
- 📊 Progress bar & live stats
- 💾 Persists data in localStorage (survives refreshes)
- 📱 Responsive layout

## Sidebar Views
- **All Tasks** — everything
- **Due Today** — tasks due today
- **Upcoming** — future tasks
- **Overdue** — past-due incomplete tasks
- **Completed** — finished tasks
- Category filters for Work, Personal, Health, Finance

## Usage
Open `index.html` in any modern browser. No server or build tools needed.

Data is saved automatically in your browser's localStorage.
